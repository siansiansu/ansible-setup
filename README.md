# Ansible-setup

Setup development environment in Ansible.

## Prerequisite

- Ansible

## Installation

1. Clone the repo

```bash
git clone git@github.com:siansiansu/ansible-setup.git
```

2. Run Ansible Playbook


```
ansible-playbook playbook.yaml
```
